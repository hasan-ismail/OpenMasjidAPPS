## 8. How to sign

Signing is handled automatically on each pull request:

1. Open Your pull request as normal.
2. The CLA bot will comment on the PR with a link to this document and ask You to
   sign **if You have not signed before**.
3. To sign, post a single comment on the PR with **exactly** this sentence:

   > I have read the CLA Document and I hereby sign the CLA

4. Your GitHub username and the signing details are then recorded in the
   `signatures/` file in the repository. You only sign **once** — future PRs are
   recognised automatically.

**under AGPL-3.0 only**: say so explicitly in Your PR description, and We will
either accept it AGPL-only or discuss an alternative. Contributions submitted
without a clear AGPL-only statement, and with a signed CLA, are made under the
full terms above.
